# ☕ Cafe Delight

A modern and responsive café website designed for coffee lovers. Cafe Delight showcases premium coffee, delicious snacks, an elegant user interface, and a seamless user experience.

## 🌟 Features

* Responsive Design
* Modern Brown & Cream Theme
* Smooth Navigation
* Hero Section
* About Section
* Menu Showcase
* Gallery Section
* Customer Reviews
* Table Reservation Form
* Mobile Friendly Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
